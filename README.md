# FoodKeeper

## Docker scripts:

By makefile:

windows MinGW32-make [command]
linux make [command] (superior system)

### make build
### make up
### make down

By docker-compose

### docker-compose up -d --build
### docker-compose up -d
### docker-compose down -v

## Available Scripts [server]

In the project directory, you can run:

### `npm run dev`
### `npm run watch`
### `npm run build`
### `npm run start`

## Available Scripts [client]

In the project directory, you can run:
### `npm start`
